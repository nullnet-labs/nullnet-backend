# Project Nullnet API Backend
Spring Boot backend application layer for the fullstack Web project [Nullnet](https://github.com/nullnet-labs).

This is a Java 21 [Spring Boot](https://spring.io/projects/spring-boot) 4.0.6 project bootstrapped with [Spring Initializr](https://start.spring.io/).

---

### Running Locally

With JDK 21 and a Maven-capable IDE installed (like Eclipse IDE for Enterprise Java and Web Developers, IntelliJ IDEA, or Spring Tool Suite) _with Lombok installed_, run this repo's contents as a Spring Boot project.

After the above, open [http://localhost:9001/v3/api-docs](http://localhost:9001/v3/api-docs) with your browser to see a basic JSON with OpenAPI info, which indicates that the server software is running.

As this application is still in early development, server features still being added.

See the [Project Nullnet organization page](https://github.com/nullnet-labs) for further documentation & repos related to this application.
